🔧 수동 확인 방법

1. 브라우저에서 직접 확인:

  • 사이트맵 접속: https://avoiderio.xyz/sitemap.xml
  • robots.txt 확인: https://avoiderio.xyz/robots.txt
  • 메인 페이지: https://avoiderio.xyz/

2. 개발자 도구 Network 탭 확인:

  1. F12 키를 눌러 개발자 도구 열기
  2. Network 탭 선택
  3. 사이트맵 URL 접속
  4. 응답 헤더와 상태 코드 확인

3. 명령어로 확인 (터미널/cmd):

# 헤더 정보 확인
curl -I https://avoiderio.xyz/sitemap.xml

# 내용 확인
curl https://avoiderio.xyz/sitemap.xml

# DNS 확인
nslookup avoiderio.xyz
